Detailed Description

Un command (fiche de presta) est un ensemble de taches que l'on veut piloter a l'aide d'indicateurs (cout, delai, qualite, avancement)

un command peut contenir des taches précises (mantis) mais également définir des objectifs d'ordre global ou non liés au dev.

un command est provisionné d'un certain budget, négocié avec le client. le cout de l'ensemble des taches devrait etre a l'equilibre avec ce budget.
